## Guide de contribution

### Récupération du code
Le code source est disponible sur GitHub. Voici comment configurer votre environnement :

1.  **Cloner le dépôt** :
    ```bash
    git clone https://github.com/votre-utilisateur/SiteJeuMdP.git
    ```
2.  **Naviguer dans le dossier** :
    ```bash
    cd SiteJeuMdP
    ```

### Installation et Lancement
1.  **Installer les dépendances** :
    ```bash
    npm install
    ```
2.  **Lancer le serveur de développement** :
    ```bash
    ng serve
    ```
    Ouvrez votre navigateur à l'adresse `http://localhost:4200/`. L'application se rechargera automatiquement à chaque modification des fichiers.

### Modifier le code
Ce projet utilise [Angular CLI](https://github.com/angular/angular-cli).
- **Générer un nouveau composant** : `ng generate component nom-composant`
- **Construire pour la production** : `ng build` (les fichiers compilés seront dans le dossier `dist/`)
